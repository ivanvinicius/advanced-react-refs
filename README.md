# Advanced Refs

## Sobre o projeto 

Apredendo a usar refs de forma avançada no React
`useImperativeHandle`
`forwardRef`

## Tecnologias utilizadas

 * [React.JS](https://reactjs.org/)

## Instalação

1. Baixe as depedências do projeto com o comando `$ yarn`.
2. Inicie o projeto com o comando `$ yarn start`.
